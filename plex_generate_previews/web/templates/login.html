How do I find my token?

Check /config/auth.json in your config volume
Or run: docker logs <container> for the file path
Or set WEB_AUTH_TOKEN in your environment/docker-compose
First time? More details…
A random token was generated on first startup and saved to /config/auth.json inside the container. This maps to whichever host path you mounted as /config.

To use your own password instead, add this to your docker-compose.yml:
environment:
  - WEB_AUTH_TOKEN=your-password
Then restart the container.
