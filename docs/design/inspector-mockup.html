07 Edge cases

Be honest when there's no signal.

Some content has no recurring theme (reality TV), unique cold opens per episode (mini-series), or no peer to compare against (single-season prestige). The Inspector tells the user clearly, rather than synthesising a low-confidence guess.

✕

REJECT — Reality TV

Survivor / Real Housewives / MAFS — no recurring intro audio. We don't try. Inspector shows "no markers detected" with a category-specific explanation.

~

Partial — Mini-series

Chernobyl / Watchmen — chromaprint fails (unique cold opens) but TheIntroDB has data 60% of the time. Surface Tier 1 hit with confidence.

🔒

Locked — manual edit

You nudged S01E03's intro end. Re-detection runs anyway, but produces a parallel candidate. Inspector shows the diff; you choose.

⇄

Bridge — Plex → Jellyfin

Plex Pass user wants Jellyfin to skip too. Read Plex's markers via plexapi, push to Jellyfin via our plugin. One click. No re-detection.

⚠

Rate limit — TheIntroDB

500 lookups/day burned. We don't block detection — fall back to Tier 2/3 chromaprint immediately. Inspector tells you when quota resets.

×

Plugin missing — Jellyfin

Read-only mode. The Inspector still shows everything we detected; "Apply" button replaced by "Install MediaPreviewBridge" with paste-ready manifest URL.
