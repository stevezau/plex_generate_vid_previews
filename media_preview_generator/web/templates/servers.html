Worked example + when you'd need this

Suppose your server stores a file at /data_16tb/movies/Inception.mkv but this app's container mounts that disk as /mnt/plex/movies/Inception.mkv — the server tells this app "the file is at /data_16tb/...", but this app can't open that path because it doesn't exist inside the container. Add a row:

  • Path on media server: /data_16tb/movies
  • Local path on this app: /mnt/plex/movies

Path the webhook sends (third column) is for the rare case where Sonarr / Radarr / Plex Direct sends yet a third path because the media is behind a union mount (MergerFS or similar). Leave blank if your webhooks already send the same paths the server uses.
